### About
This is an interactive site developed for Data Visualization Course of Federal University of Ceará, which aims to **promote insights for the community of developers** on the labor market in Information Technology, more specifically in **software development**, giving information about the **fields of expertise** of the developers (with their profiles), and the **technologies** that involves these areas though history count.
A graph is also shown where can be viewed **programming languages and their relationships** with each other, in other words, how close they are connected and related.

### Technologies
The technologies used to build this website were the basics for web development: **HTML, CSS and JavaScript**, with Github pages as host.  
Besides that, to facilitate the graph and graphics construction, the libraries **D3 and DC** were used.
**AOS.js** library was also used to program the dynamic element appearing.
