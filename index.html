Gerando insights para devs

Um apanhado geral sobre áreas da programação e suas tecnologias.

Público Alvo

  O público alvo do projeto são desenvolvedores que buscam informações sobre o mercado de trabalho na área de TI. As visualizações também são destinadas para aqueles que possuem pouco conhecimento mas sentem interesse na área.

Descrição do Projeto

    Esse é um site interativo desenvolvido para a disciplina de Visualização de Dados da Universidade Federal do Ceará, que tem como objetivo promover insights para a comunidade de desenvolvedores sobre o mercado de trabalho em Tecnologia da Informação, mais especificamente em desenvolvimento de software, fornecendo informações sobre as áreas de atuação dos desenvolvedores (juntamente com o seu perfil) e as tecnologias que envolvem essa área através da contagem de histórias.

    Um grafo também é mostrado onde pode-se visualizar linguagens de programação e seus relacionamentos umas com as outras, ou seja, como e quão próximo elas estão conectadas e relacionadas entre si.

Tecnologias/Ferramentas utilizadas

    As tecnologias que foram utilizadas usar para construir o projeto são as básicas para desenvolvimento web: HTML, CSS e JavaScript, com hospedagem no GitHub pages. Além disso, para facilitar geração interativa dos gráficos e do grafo, as bibliotecas d3 e dc foram usadas na geração dos scripts. Também a biblioteca Animate on Scroll (AOS) para programar o surgimento dinâmico dos elementos.

Os impactos na remuneração

Sobre

Uma das primeiras dúvidas que surgem naqueles que desejam se aventurar no mercado de desenvolvimento é sobre que caminho trilhar para se consolidar como profissional. Uma decisão importante é que tipo de formação se deve ter e um questionamento pertinente é onde trabalhar, utilizando o salário como parâmetro.

Para solucionar algumas dessas questões, abaixo está um grafíco de linha que relaciona o tamanho da empresa que o desenvolvedor trabalha com a média salarial anual em doláres americanos, dividindo-se entre as formações de bacharel, mestre e outros, que inclui principalmente quem não tem formação superior.

Conexão de Linguagens e Ferramentas

Sobre

Com tantas tecnologias e ferramentas disponíveis hoje para os desenvolvedores, é muito difícil decidir qual será o próximo passo para aprender uma nova tecnologia. Uma solução para isso pode ser querer aprender aquela tecnologia que se relaciona com as que você já possui mais conhecimento para, assim, ter um maior aperfeiçoamento naquilo que já trabalha.

Com isso em mente, abaixo está um grafo que relaciona e separa em grupos esse mundo de ferramentas disponíveis para os desenvolvedores. Com essa visualização, o desenvolvedor pode buscar aprender uma linguagem ou ferramenta relacionada com uma que já possui conhecimento.

Insights

O grafo é divido em 14 grupos separados por cor, como justificado na aba de Agrupamento e o tamanho de cada elemento está relacionado diretamente a quantidade de conexões que ele tem.

Assim, podemos tirar algumas conclusões facilmente: como percebendo que JavaScript é a linguagem com mais ligações e que está bastante (mas não somente) relacionada com linguagem e ferramentas front-end. Também percebemos facilmente todas as tecnlogias da Microsoft relacionadas (C#, .NET, Xamarin, VB.NET, SQL Server, etc). Também podemos ver as ligações dos Sistemas operacionais (Osx, Windows e Linux) e constatar que o Linux é a que mais possui conexões.

Agrupamento

O grafo está dividido em 14 grupos (separados por cor) e esse agrupamento foi feito através de uma técnica chamada Short Random Walks.

Essa técnica consiste em um algoritmo que busca achar clusters (comunidades) dentro de um grafo através de passeios aleatórios. A ideia é que pequenas caminhadas aleatórias dentro de um grafo tendem a permanecer na mesma comunidade.
